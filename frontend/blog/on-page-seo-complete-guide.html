On-Page SEO Complete Guide

Optimize every page for maximum search visibility

60 min read Updated Jan 20, 2025

On-page SEO (also known as on-site SEO) is the practice of optimizing individual web pages to rank higher and earn more relevant traffic in search engines. Unlike off-page SEO which focuses on external signals, on-page SEO involves optimizing both the content and HTML source code of a page. This comprehensive guide covers every aspect of on-page optimization you need to master.

Title Tag Optimization

The title tag is the most important on-page SEO element. It appears in search results, browser tabs, and social media shares.

Title Tag Best Practices

Length

50-60 characters optimal (600 pixels)

Keyword Placement

Primary keyword near the beginning

Branding

Include brand name at the end

Uniqueness

Every page needs a unique title

Good Title Tag

Best Running Shoes for Flat Feet | Nike Official

Clear, keyword-rich, branded

Poor Title Tag

Running Shoes Walking Shoes Tennis Shoes Sneakers Athletics Sports Fitness Jogging

Keyword stuffing, too long

Title Tag Formula

Follow this structure for optimal results:

[Primary Keyword] + [Secondary Keyword] | [Brand Name]

Meta Description Optimization

While not a direct ranking factor, meta descriptions significantly impact click-through rates from search results.

Meta Description Checklist

150-160 characters optimal
Include target keyword naturally
Write compelling copy
Include a call-to-action
Match search intent
Highlight unique value proposition
Use active voice

Good Meta Description

"Discover the best running shoes for flat feet. Our expert reviews feature top picks with superior arch support. Shop now and enjoy free shipping on orders over $100."

Compelling, keyword-rich, includes CTA

Poor Meta Description

"Welcome to our website. We have many products for you to look at. Click on the links to see more information about our running shoes and other items."

Generic, no keywords, no CTA

Heading Structure (H1-H6)

Proper heading hierarchy helps both users and search engines understand your content structure.

Heading Hierarchy Best Practices

H1

Main Page Title

One per page, includes primary keyword

H2

Main Section Headers

Major topics, secondary keywords

H3

Subsection Headers

Supporting points under H2s

H4-H6

Nested Subsections

Use as needed for deeper organization

Common Heading Mistakes

  • Skipping heading levels (H1 → H3)
  • Using multiple H1 tags on one page
  • Choosing style over hierarchy
  • Not including keywords in headings
  • Headings that don't describe the content

Content Optimization

Quality content is the foundation of on-page SEO. Here's how to optimize your content effectively.

Content Quality Factors

  • Original and unique content
  • Comprehensive coverage of topic
  • Accurate and up-to-date information
  • Readable and well-structured
  • Adds unique value vs competitors
  • Easily scannable with subheadings

Content Length Guidelines

Blog Posts 1,000-2,500 words
Pillar Pages 3,000-5,000 words
Product Descriptions 300-500 words
Category Pages 300+ words

Readability Optimization

Paragraph Length

2-3 sentences max

Sentence Length

Under 20 words

Font Size

Minimum 16px body text

Where to Place Keywords

1

Highest priority location

2

3

H1 Header

Main page heading

4

H2 Subheaders

Descriptive, keyword-rich

Avoid Keyword Stuffing

Natural keyword usage is essential. Over-optimization can hurt your rankings:

Good Keyword Density

1-2% naturally throughout content

Use Variations

LSI keywords and synonyms

Image Optimization

Images can drive significant traffic through Google Images and improve overall page quality.

Alt Text Best Practices

  • Describe the image accurately
  • Include relevant keywords naturally
  • Keep under 125 characters
  • Use for decorative images: leave alt=""
  • Be specific: "red running shoes on track"

Image File Optimization

  • Use descriptive filenames
  • Compress images (WebP preferred)
  • Implement lazy loading
  • Use srcset for responsive images
  • Add image sitemap for Google

Responsive Images Example

<img src="image-800.jpg"
    srcset="image-400.jpg 400w,
          image-800.jpg 800w,
          image-1200.jpg 1200w"
    sizes="(max-width: 600px) 100vw, 50vw"
    alt="Descriptive alt text">

URL Optimization

Optimized URLs

  • example.com/running-shoes-men
  • example.com/blog/seo-tips-2024
  • example.com/products/blue-coffee-mug

Poor URLs

  • example.com/page?id=12345
  • example.com/PROD_001_BLUE_MUG_XL
  • example.com/blog/post-title-with-way-too-many-words-in-it-2024

URL Best Practices

  • Use lowercase letters
  • Separate words with hyphens
  • Keep URLs short (under 60 characters)
  • Include target keywords
  • Avoid special characters
  • Remove stop words when appropriate
  • Match the page title closely
  • Use HTTPS protocol

Internal Linking Strategy

Internal links help distribute page authority and help users navigate your content.

Contextual Links

Links within content to related pages

Navigation Links

Menu and footer links to important pages

Breadcrumbs

Hierarchical navigation trail

Internal Link Best Practices

  • Use descriptive anchor text
  • Link to relevant content only
  • Vary anchor text naturally
  • Link from high-authority pages
  • Fix broken internal links
  • Avoid excessive links per page
  • Create content hubs/clusters
  • Use dofollow for internal links

External Linking

Linking to authoritative external sources can boost your credibility and SEO.

Why Link Externally?

  • Shows you've done your research
  • Builds relationships with other sites
  • Provides additional value to readers
  • Google uses it for context (E-E-A-T)
  • Can lead to reciprocal links
  • Improves content quality signals

External Link Best Practices

  • Link to authoritative, trustworthy sources
  • Use relevant, natural anchor text
  • Open external links in new tabs (target="_blank")
  • Add rel="nofollow" for untrusted or paid links
  • Limit external links (2-5 per 1,000 words)
  • Regularly check for broken external links

Page Speed & Core Web Vitals

LCP

Loading

Largest Contentful Paint

Target: Under 2.5s

FID

Interactivity

First Input Delay

Target: Under 100ms

CLS

Stability

Cumulative Layout Shift

Target: Under 0.1

Page Speed Optimization Tips

  • Optimize and compress images
  • Minify CSS and JavaScript
  • Enable browser caching
  • Use a CDN for assets
  • Defer non-critical JavaScript
  • Implement lazy loading
  • Reduce server response time
  • Minimize HTTP requests

Mobile Optimization

Mobile-First Indexing

Google primarily uses the mobile version of your site for indexing and ranking.

  • Ensure mobile content equals desktop content
  • Use responsive design (recommended)
  • Optimize touch targets (min 44x44px)
  • Readable text without zooming

User Experience Signals

Engagement Metrics

  • Dwell time (time on page)
  • Bounce rate
  • Pages per session
  • Click-through rate

UX Best Practices

  • Clear navigation structure
  • Fast loading times
  • Readable fonts and sizing
  • Intuitive internal linking
