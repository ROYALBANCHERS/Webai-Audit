What is Google Analytics 4?

What's New in GA4? (And Why It Matters)

So, what makes GA4 different from good old Universal Analytics? Google completely rebuilt the platform from the ground up. Here's what that means for you:

  • Event-based tracking - everything's an event now, giving you way more flexibility
  • Cross-platform smarts - track users seamlessly across web and mobile apps
  • Privacy built in - IP anonymization is on by default (cookieless future, anyone?)
  • Predictive powers - machine learning tells you who's likely to buy
  • Free BigQuery export - yes, really. No premium subscription needed
  • Custom funnels - see exactly where users drop off in your journey

Event-Based

Everything is an event, giving you more flexibility in tracking

AI-Powered

Predictive metrics like purchase probability

Privacy-First

Designed for a cookieless future

Getting GA4 Up and Running

Let's Get You Set Up (It's Easier Than You Think)

1

Create Your Account

Head over to analytics.google.com and sign in with your Google account

2

Set Up Your GA4 Property

Click "Create Property" and make sure you pick "GA4" (the old UA stuff is gone now)

3

Add Your Data Stream

Pop in your website URL and give it a name - Web, iOS, or Android

4

Drop in the Tracking Code

Copy that measurement ID (it looks like G-XXXXXXXXXX) and add it to your site

5

Check That It's Working

Hop into the Realtime report - if you see data popping up, you're golden!

Making the Most of Event Tracking

Here's the thing about GA4 - everything revolves around events. Seriously, everything. And that's actually great news because it gives you so much more control over what you track.

The Stuff GA4 Tracks Automatically

Good news - GA4's got your back on quite a bit:

  • • page_view (when you turn on Enhanced Measurement)
  • • session_start
  • • first_visit
  • • user_engagement
  • • scroll (how far people get down your page)
  • • click (outbound links)
  • • file_download
  • • video_start, video_complete

Events You'll Want to Add Yourself

These are the money-makers you should definitely track:

  • • search (when people use your site search)
  • • generate_lead (form fills, anyone?)
  • • login / sign_up
  • • purchase / begin_checkout
  • • add_to_cart / remove_from_cart
  • • view_item / select_item
  • • share (social shares)
  • • ad_click / ad_view

Want to Track Something Custom? Here's How

Got something specific you want to track? Say, a button click in a particular spot? You can do that with gtag.js:

gtag('event', 'custom_button_click', {
  'button_location': 'homepage_hero',
  'button_text': 'Get Started',
  'page_category': 'marketing'
});

Going Beyond the Basics with Custom Dimensions

GA4 gives you a bunch of data out of the box, but sometimes you need to track something specific to YOUR business. That's where custom dimensions come in clutch.

Event-Scoped Dimensions

Values specific to each event:

  • • Page category
  • • Author name
  • • Content type
  • • Button label

User-Scoped Dimensions

Values tied to individual users:

  • • Customer tier
  • • Subscription status
  • • Lead quality score
  • • Account type

Creating Custom Dimensions

  1. 1. Go to Configure > Custom definitions
  2. 2. Click "Create custom dimension"
  3. 3. Choose scope (event or user)
  4. 4. Set up event parameter or user property

Custom Metrics Examples

  • • Scroll depth percentage
  • • Time on page (seconds)
  • • Video completion rate
  • • Cart value

Tracking What Actually Matters: Conversions

Marking Events as Conversions

Any event can be marked as a conversion. Follow these steps:

  1. 1. Go to Configure > Events
  2. 2. Find the event you want to track as a conversion
  3. 3. Toggle the "Mark as conversion" switch
  4. 4. The event will now appear in conversion reports

Lead Generation

form_submit, click_to_call

E-commerce

purchase, add_to_cart

Content Engagement

file_download, scroll_90

Monetization Reports

E-commerce Setup Requirements

To see monetization reports, you need to:

  1. 1. Enable e-commerce in your data stream settings
  2. 2. Send e-commerce events (purchase, refund, etc.)
  3. 3. Include required parameters (items, transaction_id, value)

Overview

Total revenue, orders, average order value

Items

Product revenue, quantity, refunds

Purchasers

Customer LTV, repeat purchase rate

Diving Deep with the Explore Section

Ready to graduate from the standard reports? The Explore section is like having a data science lab built right into GA4. It's where you can ask specific questions and get real answers.

Explore Templates

  • • Path Exploration: User journeys through your site
  • • Funnel Exploration: Steps to conversion
  • • Segment Overlap: Audience intersections
  • • Cohort Exploration: Retention analysis
  • • User Lifetime: LTV over time
  • • Segment Match: Audience characteristics

Custom Explorations

Build your own analysis with:

  • • Techniques: Explore, Funnel, Path
  • • Dimensions: Segment by any attribute
  • • Metrics: Choose what to measure
  • • Filters: Narrow your data
  • • Comparisons: A/B test segments

Linking Google Search Console

Benefits of Linking

  • See search query data
  • View organic landing pages
  • Access country/region data
  • See search result position data
  • Compare organic vs paid
  • Access impression data
  • View click-through rates
  • Improve SEO analysis

How to Link Search Console

  1. 1. In GA4, go to Admin > Property settings
  2. 2. Scroll to "Search Console" section
  3. 3. Click "Link Search Console"
  4. 4. Select your property from the list
  5. 5. Submit and confirm the linking
