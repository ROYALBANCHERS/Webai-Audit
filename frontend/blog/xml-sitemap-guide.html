Complete XML Sitemap Guide

Master sitemaps for better crawling and indexing

25 min read Updated Dec 30, 2024

XML sitemaps are roadmaps for search engines, helping them discover and understand your website's structure. A well-optimized sitemap ensures all your important pages are crawled and indexed. This comprehensive guide covers everything you need to know about creating, optimizing, and maintaining XML sitemaps.

What is an XML Sitemap?

An XML sitemap is a file that lists your website's important pages, helping search engines discover and crawl them more efficiently:

Crawl Discovery

Helps search engines find all your pages

Faster Indexing

New content gets indexed quicker

Page Metadata

Provides lastmod and priority info

Types of Sitemaps

XML Sitemap

The standard format for most websites. Lists URLs with optional metadata.

sitemap.xml

News Sitemap

For news sites. Lists articles with publication dates.

news-sitemap.xml

Video Sitemap

For video content. Includes thumbnails and duration.

video-sitemap.xml

Image Sitemap

For image-heavy sites. Includes licenses and geo locations.

image-sitemap.xml

Creating an XML Sitemap

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">

<url>
<loc>https://example.com/</loc>
<lastmod>2024-12-30</lastmod>
<changefreq>weekly</changefreq>
<priority>1.0</priority>
</url>

<url>
<loc>https://example.com/about</loc>
<lastmod>2024-12-28</lastmod>
<changefreq>monthly</changefreq>
<priority>0.8</priority>
</url>

</urlset>

Sitemap Tags Explained

<loc>

The page URL (required)

<lastmod>

Last modification date (optional)

<changefreq>

How often page changes (always, hourly, daily, weekly, monthly, yearly, never)

<priority>

0.0 to 1.0 relative importance

Sitemap Best Practices

Do's

  • Include all important pages (canonical URLs only)
  • Keep sitemaps under 50MB and 50,000 URLs
  • Split large sitemaps into multiple files with a sitemap index
  • Update sitemaps when adding new content
  • Place sitemap.xml in your website root

Don'ts

  • Don't include redirect URLs, canonicalized pages, or noindex pages
  • Don't include admin pages, login pages, or search results
  • Don't mix HTTP and HTTPS URLs in the same sitemap
  • Don't include session IDs in URLs

Sitemap Index Files

For sites with many URLs, use a sitemap index:

<sitemapindex xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
<sitemap>
<loc>https://example.com/sitemap1.xml</loc>
</sitemap>
<sitemap>
<loc>https://example.com/sitemap2.xml</loc>
</sitemap>
</sitemapindex>

Submitting to Search Engines

Google Search Console

  1. Add your property to GSC
  2. Go to "Sitemaps" section
  3. Enter your sitemap URL
  4. Click "Submit"
https://search.google.com/search-console

Bing Webmaster Tools

  1. Add your site to Bing Webmaster
  2. Go to "Sitemaps" section
  3. Submit your sitemap
https://www.bing.com/webmasters

robots.txt Location: Also reference your sitemap in robots.txt:

Sitemap: https://example.com/sitemap.xml
